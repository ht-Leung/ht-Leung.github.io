# Haotian Liang Homepage

这是一个用于个人主页和项目展示的静态网站。

## 本地预览

可以直接打开 `index.html`，也可以在当前目录启动本地静态服务：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## GitHub Pages 部署

将仓库推送到 GitHub 后，在仓库设置中启用 GitHub Pages：

1. 打开仓库的 `Settings`。
2. 进入 `Pages`。
3. 将部署来源设置为 `main` 分支。
4. 目录选择仓库根目录。

如果仓库名为 `ht-Leung.github.io`，启用后主页地址通常是：

```text
https://ht-leung.github.io/
```
